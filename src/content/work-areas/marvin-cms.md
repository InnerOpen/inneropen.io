Marvin CMS is a modern headless content management system designed for technical teams. It provides schema-driven content modeling, automatic API generation, event-based integrations, and a flexible plugin architecture.

The Marvin project family includes:

- **Marvin** - Core CMS engine
- **Marvin SDK** - Python SDK for programmatic access
- **Marvin CLI** - Command-line interface for operations
- **Marvin Node** - Node.js SDK and type definitions
- **Marvin Plugin Template** - Scaffold for building plugins

## Focus Areas

- Content modeling and schema design
- API-first architecture
- Event-driven workflows
- Plugin extensibility
- Developer experience
