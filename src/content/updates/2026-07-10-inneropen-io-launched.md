The InnerOpen.io engineering site is now live. This site serves as the technical home for InnerOpen's open engineering work.

## What's Here

**Work Areas** - InnerOpen works across multiple technical areas:
- Marvin CMS and related projects
- OpenShift platform engineering
- Kubernetes operations
- Platform engineering and developer experience
- Automation and AI-assisted infrastructure

**Updates** - Release announcements, milestones, and project updates from InnerOpen repositories.

**Engineering Notes** - Short-form technical writing on platform engineering, automation, modernization, and operational patterns.

## Publishing Model

External InnerOpen repositories can publish updates to this site via automated workflows. Updates are submitted as pull requests for human review before publication, ensuring quality while enabling frequent communication.

## Machine-Readable Feeds

The site exposes RSS and JSON feeds that can be consumed by other systems, enabling future integration with the main InnerOpen.com consulting site.

## What's Next

- Publishing workflows for Marvin repositories
- Additional engineering notes on OpenShift and Kubernetes work
- Reference architectures and migration playbooks
- Automation templates and reusable patterns

This site is built with Astro, uses content collections for type-safe content management, and deploys to GitHub Pages. All code is open source.
